# QuickPrint-UKM
QuickPrint offers a self-service, digital system where students can print without visiting the shop. Features include online document upload, campus delivery, cashless payment, and a reward system. This improves convenience, reduces staff burden, and balances resource use throughout peak and off-peak periods.
